⚙️ Backend 기술 스택

FastAPI

  • FastAPI를 활용하여 기본적인 REST API 엔드포인트 개발 및 경로 작업(라우팅) 가능
  • Pydantic을 이용한 데이터 유효성 검사 및 기본적인 비동기 처리 로직 구현 가능

SpringBoot

  • SpringBoot 기반의 웹 애플리케이션 개발 및 RESTful API 설계/구축 경험 보유
  • JPA/Hibernate를 이용한 데이터베이스 연동 및 기본적인 의존성 주입(DI), 보안 설정 적용 가능
