🗄️ Database 기술 스택

MySQL

  • 관계형 데이터베이스 모델링 기본 개념을 이해하고 SQL을 사용하여 데이터 조회 및 관리 가능
  • 간단한 테이블 설계 및 인덱스 활용 경험 보유

MongoDB

  • NoSQL 및 Document 기반 데이터베이스 개념을 이해하고 기본적인 CRUD 작업 수행 가능
  • Collection 설계 및 기본적인 쿼리 작성 경험 보유

Redis

  • In-Memory 데이터 구조 저장소의 개념을 이해하고 캐싱, 세션 관리 등에 활용 가능
  • 다양한 데이터 타입(String, List, Set 등) 사용 및 기본 명령어 활용 경험 보유

Firestore

  • Firebase Console에서 제공하는 서비스(FCM, Realtime DB, Analytics) 사용 경험 있음
  • 데이터 모델링 및 보안 규칙 설정 가능
